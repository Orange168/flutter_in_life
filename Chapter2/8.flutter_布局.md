# 线性布局

## Row

```dart
Row({
  ...  
  TextDirection textDirection,    
  MainAxisSize mainAxisSize = MainAxisSize.max, //方向占用的空间   
  VerticalDirection verticalDirection = VerticalDirection.down,  //垂直的对齐方向从上到下
  MainAxisAlignment mainAxisAlignment = MainAxisAlignment.start, //水平对齐
  CrossAxisAlignment crossAxisAlignment = CrossAxisAlignment.center, //垂直对齐
  List<Widget> children = const <Widget>[],
})
```
